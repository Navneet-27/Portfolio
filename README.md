# Portfolio Website

This is a simple static portfolio website created based on Navneet Sinha's resume. It showcases personal information, skills, projects, experience, and contact details.

## Structure
- `index.html` &ndash; main page with all sections.
- `css/style.css` &ndash; basic styling.

## Usage
1. Open `index.html` in any web browser to view the portfolio.
2. Modify content or styles as needed.

## Deployment
You can host this folder on GitHub Pages, Netlify, or any static file server.

Feel free to extend the site with additional styling, sections, or interactivity.
